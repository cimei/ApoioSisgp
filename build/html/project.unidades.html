Unidades (views)

As Unidades na estrutura da instituição. Este sistema restringe os tipos aos valores (1,”Instituição”),(2,”Diretoria”),(3,”Coordenação-Geral”),(4,”Coordenação”),(5,”Serviço”),(6,”Outro”). Para situação, ou é (1,”Ativa”), ou (2,”Inativa”).
